# ATTiny85 / Digispark Referenz

Nicht als Primärlösung gedacht. Dieses Verzeichnis ist nur als Erinnerung an die diskutierte Minimalvariante enthalten.

Gründe gegen ATTiny85 als Hauptarchitektur:

- Software-USB statt echter USB-Controller
- wenig RAM
- eingeschränkte Erweiterbarkeit
- weniger robust für mehrkanalige Dauerlösung
